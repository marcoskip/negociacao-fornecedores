# SPEC 0.1 — Gestão de Fornecedores

## Identificação

- **ID:** `0.1`
- **Nome:** Gestão de Fornecedores
- **Contexto:** `varejista`
- **URL:** `ipa.app/fornecedores`
- **Largura:** 1280px
- **Persona:** Comprador do varejista (gerencia a rede de fornecedores)
- **Quando aparece:** Acesso direto pelo menu lateral (item "Fornecedores"). Também é o destino pós-envio de convite (0.2 → 0.1).
- **Wireframe:** `./0.1.png`

## Objetivo

Visualizar todos os fornecedores vinculados ao varejista, filtrar por status/atividade, e iniciar o convite de um novo fornecedor.

### Colunas da tabela (ordem e largura)

Grid 7 colunas: `2fr 1.3fr 1.3fr 1.1fr 0.8fr 1fr 60px`

| Coluna | Conteúdo | Sortable |
|---|---|---|
| EMPRESA | Nome da empresa (negrito) | não |
| CNPJ | CNPJ formatado (fonte mono) | não |
| REPRESENTANTE | Nome do contato. Mostrar "—" se sem representante (status pendente) | não |
| STATUS | Pill de status do vínculo | não |
| NEGOCIAÇÕES | Inteiro (fonte mono) | **sim** (asc/desc) |
| ÚLTIMA ATIV. | Data relativa (ver COPY.md) | **sim** (asc/desc) |
| AÇÕES | Botão ⋯ | não |

### Ordenação default

Sem coluna ativa → ordena por **ÚLTIMA ATIV. descendente** (mais recente primeiro).

### Menus contextuais (botão ⋯)

| Status do fornecedor | Itens do menu |
|---|---|
| Ativo | "Ver negociações" · "Editar contato" · "Inativar vínculo" |
| Convite pendente | "Reenviar convite" · "Copiar link" · "Cancelar convite" |
| Inativo | "Reativar vínculo" · "Remover" (em vermelho de alerta) |
